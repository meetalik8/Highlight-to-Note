# Highlight-to-Note
A simple Chrome Extension that lets you highlight any text on the web and save it instantly.
All your saved notes appear inside the extension popup, where you can easily copy or delete them!

# How to Install:
1. Download this repository as a ZIP
2. Extract the folder
3. Open Chrome and go to: chrome://extensions/
4. Enable Developer Mode (top-right toggle)
5. Click Load Unpacked
6. Select the extracted folder
7. Extension is now installed!

# How to Use:
1. Highlight any text on a webpage
2. Right-click and select “Save to Highlight-to-Note”
3. Open the extension popup
4. View, copy, or delete your saved notes

# Privacy:
This extension does not collect any personal data.
All notes are stored only using Chrome’s storage.sync and remain on your device/account.
